STRUCTURAL CHANGES
------------------

1. STATIC HTML / NO CMS
   Site is vanilla HTML/CSS/JS — no WordPress, no page builder.
   Result: load times significantly faster than WordPress equivalents.
   Pingdom: ~427ms / ~1MB at time of last test.

2. AVIF IMAGE CONVERSION
   All 120+ images converted from JPG to AVIF using avifenc (libavif).
   Significant file size reduction across the board.

3. INLINE SVG SPRITE
   Replaced 486KB Lucide CDN script with a ~2KB inline SVG sprite.
   19 icons defined as <symbol> elements, referenced via <use href>.

4. BEFORE/AFTER SLIDERS
   Two sliders on homepage (roof, fence) using CSS clip-path reveal.
   Pointer Events API with setPointerCapture for smooth drag.
   touch-action: none on handle prevents browser scroll conflict.

5. MOBILE MENU
   Hamburger overlay with scroll lock, Escape key support, aria-expanded.

6. ESTIMATE MODAL
   Single modal shared across all CTAs per page.
   Each page uses a unique Netlify form name to distinguish submissions:
   estimate-modal, estimate-modal-replacement, estimate-modal-repair.
   Same pattern for footer forms.

7. NETLIFY FORMS
   All forms use data-netlify="true" and honeypot fields.
   Each form has a unique name attribute so submissions are identifiable
   by which page they came from.

8. TRUST SECTION (HOMEPAGE)
   Two separate trust-bar sections merged into one 6-card section.
   Deduplicated overlapping "owner" cards — each of the 6 cards now
   covers a distinct trust signal:
   - Owner on Every Job (job oversight)
   - No Sales Rep / No Commission (pricing honesty)
   - Permits Pulled (regulatory compliance)
   - Referral-Driven Reputation (social proof)
   - Owens Corning — Built Nearby (product trust + local connection)
   - We Like Those Odds (confidence / competitive angle)

9. PARTNERS SECTION
   Desktop: animated infinite scroll marquee.
   Mobile: 2-column static grid — all logos visible without scrolling.
   NOTE: GAF Certified and Lowe's Installer logos are placeholders.
   Craig needs to confirm which accreditations he actually holds before
   these go live permanently.

10. FEATURE PHOTO (INNER PAGES)
    Mini-gallery (6-photo grid) replaced with single full-width photo
    at 16:6 aspect ratio on roof-replacement.html and roof-repair.html.

11. GREEN ACCENT ELEMENTS (INNER PAGES)
    Em-dash (—) in green added above each feature-item title via CSS ::before.
    Prose-accent class: subtle green left border on body paragraphs in
    the dark CRC Difference section.

12. CITY BADGE HOVER
    On hover: badge background turns green, text and map icon both turn white.
    Previously icon stayed green (invisible against green background).

13. BEFORE/AFTER SLIDERS — FENCE AND ROOF
    Images: Images/before_afters/roof_before.avif, roof_after.avif,
    fence_before.avif, fence_after.avif.

14. STATUS DOT
    Green pulsing dot next to phone number in nav.
    Turns grey (offline class) outside Mon–Sat 8AM–6PM via JS.

15. ANIMATED COUNTERS
    Three stats in trust bar: 16+ Years / 26 Reviews / 100% Free Inspections.
    Animate on scroll using IntersectionObserver. Respects prefers-reduced-motion.

16. SCROLL FADE-UP ANIMATIONS

    Disabled entirely when prefers-reduced-motion is set.
